# Highway Safety & Incident Analysis (Autostrada A4)

## Overview
This repository contains a comprehensive **Exploratory Data Analysis (EDA)** and statistical evaluation of traffic incidents along **Autostrada A4** (Turin - Trieste corridor). 

By processing a dataset of over 4,700+ highway crash records, this project aims to uncover the primary drivers of crash severity, temporal distribution of accidents, and the impact of environmental risk factors on road safety. The insights generated are designed to support highway operators in deploying targeted preventative measures and optimizing emergency response allocation.

## Key Analytical Workflows
1. **Data Wrangling & Cleaning:**
   * Processed messy, real-world incident logs using `pandas`.
   * Handled missing values, standardized timestamp formats, and categorized weather/lighting conditions.
2. **Temporal & Spatial Distribution:**
   * Analyzed crash frequencies across different times of day, days of the week, and seasonal peaks.
   * Identified high-risk time windows (e.g., peak commute hours vs. late-night fatigue hours).
3. **Severity Profiling:**
   * Evaluated the correlation between environmental factors (rain, fog, poor illumination) and accident severity (property damage vs. fatal injuries).
4. **Data Visualization:**
   * Developed insightful visual dashboards (heatmaps, bar charts, line graphs) using `matplotlib` and `seaborn` to communicate findings clearly to non-technical stakeholders.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Domain:** Traffic Safety, Transportation Engineering, Statistical Analysis
