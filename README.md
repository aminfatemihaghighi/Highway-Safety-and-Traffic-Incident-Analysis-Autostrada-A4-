# Highway Safety and Traffic Incident Analysis (Autostrada A4)

## Project Overview
This repository contains a comprehensive road safety and traffic incident analysis project based on historical crash data from the Autostrada A4 (Torino-Trieste) in Italy. The objective of this project is to process and analyze microscopic crash records to identify safety trends, evaluate environmental risk factors, and categorize incident severity along a major European transport corridor.

## Dataset Profile
* **Data Source:** Historical highway crash database (2013–2021)
* **Volume:** 4,717 individual crash records
* **Key Variables Analyzed:** Crash severity, temporal distribution (Day/Night, Year, Month, Hour), environmental conditions (Weather, Road Surface), infrastructure features (Lane Type, Workzone Proximity), and vehicle classification (Light vs. Heavy Vehicles).

## Analytical Focus Areas
1. **Severity Classification:** Segmented accidents into Property Damage Only (PDO), Injury-causing crashes, and Fatalities to calculate overall risk distributions.
2. **Environmental & Spatial Factors:** Evaluated how weather conditions, dry/wet road surfaces, and specific highway segments impact the frequency of incidents.
3. **Data Aggregation:** Utilized Excel pivot tables and data aggregation techniques to summarize total injuries and fatalities against varying road conditions and infrastructure types.
